spring-quartz-test 
============

>** spring-quartz-test은 Spring과 Quartz 를 이용해서 만든 Java Cron 어플리케이션입니다. **

-------------
**spring-quartz-test 에 적용된 기술은 다음과 같습니다.**

사용된 오픈소스|버전
------------|---
java|1.7
spring|4.1.8
maven|3.2.3
quartz|2.2

-----
설치방법은 eclipse에서 아래와 같이 처리합니다. 
>**import > maven > Existing Maven Projects**
